Title : Grocery Delivery System

 Description:
This project is a Grocery Delivery System where users can browse grocery items, add them to cart, place orders, and track order status. Users can also make payments and provide feedback.
The admin can manage products, categories, users, and orders, ensuring smooth operation of the platform.

Model Description

ADMIN:
LOGIN
VIEW USERS
MANAGE CATEGORIES (Fruits, Vegetables, Dairy, etc.)
MANAGE PRODUCTS (Add / Update / Delete)
MANAGE ORDERS (Update Status: Pending / Shipped / Delivered)
MANAGE PAYMENTS
VIEW FEEDBACK & RATINGS

USER:
REGISTER
LOGIN
VIEW CATEGORIES
VIEW PRODUCTS
ADD TO CART
UPDATE CART (Increase / Decrease Quantity)
REMOVE FROM CART
PLACE ORDER
SELECT ADDRESS
VIEW ORDER STATUS
MAKE PAYMENT
WRITE FEEDBACK & RATING
